Gothic 1 Remake Lockpick Solver by Ainez — Instructions / Инструкция
===========================================================

[EN] QUICK START GUIDE:
-----------------------
1. Run "Gothic 1 Remake Lockpick Solver by Ainez.exe".
2. Toggle "On Top" in the header to keep the window visible over your game window.
3. Use the arrow buttons to set the current visual offsets of the lock plates (as they appear in the game).
4. Set up custom lock rules (if plates move together):
   * Click the "Rules" button in the top right.
   * If shifting a plate left also moves other plates, specify their direction of movement (left or right).
   * Click "Save" to apply rules.
5. If the lock has fewer than 7 plates (e.g., 6 plates):
   * Just leave the unused (7th) plate at "0" (centered).
   * It will not participate in the calculations.
6. Click "Find Solution".
7. Follow the step-by-step list of moves displayed at the bottom.
8. Use the playback buttons (prev / next) to visually step through the solution.
9. Click "Reset" to return all plates to zero and clear all custom lock rules.


[RU] ИНСТРУКЦИЯ ПО ПРИМЕНЕНИЮ:
-----------------------------
1. Запустите "Gothic 1 Remake Lockpick Solver by Ainez.exe".
2. Для удобства переключите в шапке режим на "Поверх" (Always on Top), чтобы окно программы всегда было поверх окна с игрой.
3. Стрелочками выставьте текущее смещение пластин замка (как в игре).
4. Настройте правила замка (если пластины сдвигаются вместе):
   * Откройте окно «Правила» (Rules) в правом верхнем углу.
   * Если при сдвиге пластины влево двигаются и другие пластины, укажите их направление смещения (влево или вправо).
   * Нажмите «Сохранить» (Save) для применения.
5. Если в замке меньше 7 пластин (например, 6 пластин):
   * Просто оставьте лишнюю (7-ю) пластину на значении "0" (посередине).
   * Она никак не будет участвовать в расчете решения.
6. Нажмите кнопку "Найти решение" (Find Solution).
7. Внизу появится пошаговый список ходов (какую пластину двигать влево или вправо).
8. Используйте кнопки плеера (Шаг пред / Шаг след), чтобы наглядно видеть изменение положений на каждом шаге.
9. Кнопка "Сброс" (Reset) обнуляет положения пластин и сбрасывает все правила.
