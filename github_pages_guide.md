# Инструкция: Как бесплатно запустить решатель в браузере через GitHub Pages

GitHub Pages позволяет бесплатно разместить ваш сайт в интернете. Никаких серверов настраивать не нужно, всё делается в пару кликов.

---

## Шаг 1. Создание репозитория на GitHub
1. Зарегистрируйтесь или войдите на [github.com](https://github.com/).
2. В правом верхнем углу нажмите кнопку **«+»** и выберите **New repository** (Новый репозиторий).
3. В поле **Repository name** введите имя проекта на английском, например:
   `gothic-lockpick-solver`
4. Выберите тип **Public** (это обязательно, чтобы сайт открывался у всех).
5. Нажмите кнопку **Create repository** (Создать репозиторий) в самом низу.

---

## Шаг 2. Загрузка файлов
1. На странице созданного репозитория найдите и нажмите ссылку **uploading an existing file** (загрузить существующий файл).
2. Перетащите в окно браузера следующие 3 файла из папки проекта:
   * `index.html`
   * `styles.css`
   * `app.js`
3. Дождитесь окончания загрузки файлов.
4. Нажмите зеленую кнопку **Commit changes** (Сохранить изменения) внизу.

---

## Шаг 3. Включение сайта (GitHub Pages)
1. В верхнем меню вашего репозитория перейдите во вкладку **Settings** (Настройки, иконка шестеренки ⚙).
2. В левом вертикальном меню найдите раздел **Code and automation** и выберите пункт **Pages**.
3. В разделе **Build and deployment** найдите блок **Source** (обычно там выбрано *Deploy from a branch*).
4. Ниже в выпадающем меню **Branch** выберите ветку `main` (или `master`) вместо *None*.
5. Нажмите кнопку **Save** (Сохранить).

---

## Шаг 4. Готово! Получение ссылки
1. Подождите около 30–60 секунд.
2. Обновите страницу настроек Pages.
3. В самом верху страницы появится блок с зеленой галочкой и готовой ссылкой на ваш сайт. Она будет выглядеть так:
   `https://ваш-никнейм.github.io/gothic-lockpick-solver/`
4. Скопируйте эту ссылку и делитесь ей на любых форумах!

---

### Альтернативный способ (запуск на ПК без интернета):
Вы и любой пользователь можете просто скачать файлы (`index.html`, `styles.css`, `app.js`) в одну папку и дважды кликнуть по файлу **`index.html`** — решатель откроется и будет полноценно работать прямо в браузере локально, даже без подключения к сети.
